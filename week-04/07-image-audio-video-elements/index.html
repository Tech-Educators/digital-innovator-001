Image, Audio and Video Elements in HTML. Woop!

Starting with Images

We can use the img tag to display static images

It's important that we use a src, alt and perhaps sometimes width and height (although we can also do this in CSS)

We can use image sources where the image url is copied from the web

However, we want control over our media. So we should locally store and host our images

We can also use the img tag to display gifs

Audio

We use the audio tag to bring audio to our page

We have to use the 'controls' attribute so that the player becomes visible

We can use the 'autoplay' attribute, however, this isn't good user experience, and generally is disabled by default

Video

We get videos to appear on our page with the ______ tag

For videos, we should have the 'controls' attribute so that the user can control the video.

We can also give it a width, or we can style the size in CSS

The 'poster' attribute lets us place a template image onto the video before it plays. Again, better user experience.

And we can also use 'autoplay' and 'muted' to further direct our video.
